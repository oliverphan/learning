All tag pairs and contents are called HTML elements. Elemnets can be body, paragraph, html... tags.

Attributes/paramters are specified in the starting tag with name/value pairs with the equal sign

You can add attributes to virtually all tags, as well as tooltips with the title tag for paragraphs. Make sure to be cognizant of your headers.

Press Ctrl + U on a webpage to view the HTML source code.

Make sure to also play around with colours in CSS!

You can even set borders and font sizes!
